## ğŸš€ Quick Start

### 1. Clone Repository
```bash
git clone https://github.com/kathir45/chaintech-network.git
cd chaintech-network
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Setup
Create `.env` file:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
```

### 4. Database Setup

```bash
npm run dev
```
